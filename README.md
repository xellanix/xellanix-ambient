## Notes for Keyboard shortcuts:
- [x] Toggle dark mode: D
- [x] Toggle shuffle: S
- [x] Toggle loop: L
- [x] Next track: N
- [x] Previous track: P
- [x] Mute/Unmute: M
- [x] Add track to playlist: A
- [x] Toggle maximize lyrics: F
- [x] Switch to playlist view: Shift + P
- [x] Switch to queue view: Shift + Q
